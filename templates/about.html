About NOLA.docs

NOLA.docs is designed to be a form-based document generation platform that gives you complete control over document formatting and output.

Key Features

✓ Locked styling for consistent output
✓ Form-controlled logic; users input data, not format
✓ Automated DOCX output
✓ User authentication with roles
✓ Admin dashboard for template management
✓ Scalable template system

Tech Stack

⚙️ Backend

  • • Flask
  • • WTForms
  • • Flask-Login
  • • SQLAlchemy
  • • python-docx-template

🎨 Frontend

  • • Modern CSS Design System
  • • Jinja2 Templates
  • • Server-side rendering
  • • Responsive design

This is a prototype application for building a robust document generation system. More features and improvements coming soon!
